Dynamic i18n

This Demo page is to showcase how i18n (internationalization) works. Key approach is to update the language change without refreshing the page. Loads locale specific contents dynamically, whenever language is changed and updating the translated content to match the selected language. Importantly, this uses custom html element so that the content update is easily done.

Select a language from the links below to change the language. Observe that the language is changed without page refresh.

Note: The content used in this demo is translated using Google translation service.
